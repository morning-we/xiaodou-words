# 小豆单词 (XiaoDou Words)

一款现代化的 Web 版单词学习应用，支持单词练习、积分系统、签到功能和完整的后台管理。

## ✨ 核心功能

### 🎯 单词练习
- 每组 20 个单词
- 每题 30 秒倒计时
- 显示单词、音标、释义
- 支持播放读音（Web Speech API）
- 3 个选项（ABC），随机排列
- 实时计分（每题 1 分）
- 答案反馈（正确/错误）

### 👤 用户系统
- 用户注册/登录
- 自动保存登录状态
- 多用户支持

### 📊 个人中心
- 总积分展示
- 总得分统计
- 正确率计算
- 积分排名
- 每日签到（+1 积分）
- 连续签到天数
- 练习记录
- 操作日志
- 客服联系

### 🛠️ 后台管理
- **菜单管理**：添加、编辑、删除单词菜单
- **单词上传**：批量上传单词，支持自定义格式
- **用户管理**：查看所有用户和排名
- **操作记录**：查看所有用户操作记录

### 💡 积分规则
- 每日签到获得 1 积分
- 单词练习总分每达到 100 分获得 1 积分
- 积分越高，排名越靠前

## 🚀 快速开始

### 前置要求
- Node.js 24+

```bash

```


### 构建生产版本
```bash

```

## 📖 使用指南

### 1. 用户注册/登录
- 首次访问会看到 5 秒启动页面（可跳过）
- 进入登录/注册页面
- 注册新用户或使用已有账户登录

### 2. 开始单词练习
- 在首页选择一个单词菜单
- 点击"开始练习"按钮
- 每题有 30 秒答题时间
- 选择正确答案获得 1 分
- 完成所有 20 题后查看结果

### 3. 个人中心
- 查看积分、得分、正确率、排名
- 每日签到获得积分
- 查看练习记录和操作日志
- 联系客服

### 4. 后台管理
- 进入"后台管理"页面
- 管理单词菜单
- 上传单词文件
- 查看用户和操作记录

## 📝 单词文件格式

上传单词时，请使用以下格式（制表符 Tab 分隔）：

```
单词	音标	释义	正确答案	选项A	选项B	选项C
abandon	/əˈbændən/	放弃；抛弃	放弃	继续	开始
ability	/əˈbɪləti/	能力；本领	能力	残疾	可能性
abnormal	/æbˈbɔːml/	不正常的；反常的	正常的	不正常的	抽象的
```

**说明：**
- 每行一个单词
- 使用制表符（Tab）分隔
- 正确答案放在第 4 列
- 3 个选项分别放在第 5、6、7 列

## 🎨 技术栈

- **框架**: Next.js 16 (App Router)
- **UI 库**: React 19 + shadcn/ui
- **样式**: Tailwind CSS 4
- **语言**: TypeScript 5
- **数据存储**: localStorage
- **语音**: Web Speech API

## 🐛 常见问题

### Q: 为什么数据刷新后还在？
A: 当前使用 localStorage 存储，数据保存在浏览器中，刷新不会丢失。

### Q: 如何清空所有数据？
A: 在浏览器开发者工具中清除 localStorage，或在后台管理中删除所有菜单和单词。

### Q: 如何添加更多单词？
A: 在后台管理的"单词上传"标签页，选择菜单并粘贴单词内容。

### Q: 语音播放不工作？
A: 确保浏览器支持 Web Speech API，并检查系统音量设置。
